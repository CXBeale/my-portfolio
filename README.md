# My Portfolio

A clean, modern personal portfolio that showcases my projects, skills, and personality.

## Description

This is a practice project to improve my skills in HTML, CSS, and JavaScript. The portfolio features a responsive design with interactive elements and smooth animations.

## Features

- Responsive navigation with mobile hamburger menu
- Hero section with animated text
- About section with skills display
- Projects showcase with filtering
- Contact form with validation
- Dark/light mode toggle
- Smooth scroll navigation
- Back to top button

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Setup

1. Clone the repository:
   ```
   git clone https://github.com/CXBeale/my-portfolio.git
   ```

2. Open `index.html` in your browser

## Project Structure

```
my-portfolio/

├── images/          # Image files
├── js/              # JavaScript files
├── index.html       # Main HTML file
```
